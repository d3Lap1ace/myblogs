# Claude Code 状态栏脚本 context-bar.sh

来源：https://github.com/ykdojo/claude-code-tips/blob/main/scripts/context-bar.sh

实时显示：模型名 | 工作目录 | git 分支/同步状态 | 上下文进度条 | 上一条消息预览

## 安装

```bash
curl -o ~/.claude/context-bar.sh \
  https://raw.githubusercontent.com/ykdojo/claude-code-tips/main/scripts/context-bar.sh
chmod +x ~/.claude/context-bar.sh
```

## 配置 statusline

在 `~/.claude/settings.json` 中添加：

```json
{
  "statusline": "bash ~/.claude/context-bar.sh"
}
```

## 换颜色

编辑脚本第 4 行：

```bash
COLOR="blue"  # 可选: gray orange blue teal green lavender rose gold slate cyan
```

## 效果示例

```
Claude Sonnet 4.6 | 📁Studio | 🔀master (0 files uncommitted, synced 5m ago) | ▄░░░░░░░░░ ~2% of 200k tokens
💬 how to use ti script
```
